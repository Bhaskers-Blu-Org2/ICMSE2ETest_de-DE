# <a name="high-dpi-support-for-it-professionals"></a>Hochauflösenden Unterstützung für IT-Experten


Windows 8.1 verfügt über neue Features, die der Endbenutzer mit Premium hohe Dichte Anzeige Bereiche zu optimieren. Wenn Sie mithilfe von Windows 8.1 Aktivitäten wie mit einem externen Anzeige und Skalierung auf dem primären Monitor Desktop projizieren aus erheblich verbessert werden Gibt vorherigen Windows frei. Den größten Nutzen aus dieser Features wird bei Verwendung eine Dichte Anzeige, beispielsweise ein 2560 x 1440 Bildschirm mit 225 DPI und Skalieren von 200 % desktop angezeigt. In diesem Thema wird erläutert, was besondere folgende Teile der Premium-Display ist, und Funktionsweise von Windows 8.1, um eine bessere Unterstützung für diese bereitzustellen. Außerdem erfahren Sie, mögliche Probleme – einschließlich Unschärfe oder Verschwommenheit anzeigen –, die möglicherweise einige Benutzer mit niedriger Dichte zeigt auf 125 % Skalierung, auswirken und wie Enterprise IT Professionals können sie in Windows 8.1 adressieren.
