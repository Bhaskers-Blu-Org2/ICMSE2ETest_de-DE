# <a name="change-the-logging-mode"></a>Ändern Sie den Protokollierungsmodus


In diesem Verfahren wird veranschaulicht, wie den Protokollierungsmodus Windows Performance Aufzeichnung (WPR) von der Benutzeroberfläche (UI) zu ändern. Informationen zum Starten eine Aufzeichnung über die Befehlszeile finden Sie unter [WPR Command-Line Options](wpr-command-line-options.md).

**So ändern Sie den Protokollierungsmodus**

1.  Klicken Sie auf **Windows-Leistung Aufzeichnung** auf dem Bildschirm Windows 8 **Starten** .

2.  Klicken Sie auf **Weitere Optionen**.

3.  Wählen Sie den Protokollierungsmodus, aus der Dropdownliste **Logging Modus** . Der Standardmodus ist **Arbeitsspeicher**.

**Vorsicht**  
Wählen Sie für mehr Aufzeichnungen **Arbeitsspeicher**. Wenn Sie **Datei**auswählen, kann die Datei sehr groß, da der verfügbare Speicherplatz die einzige Einschränkung Dateigröße liegt. Wenn die Datei ist zu groß ist, können Sie nicht analysiert werden in Windows Performance Analyzer (WPA) sein.
