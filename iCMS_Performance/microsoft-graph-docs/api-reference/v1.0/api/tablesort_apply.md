# <a name="tablesort-apply"></a>TableSort: anwenden

Führen Sie einen Sortiervorgang.
## <a name="prerequisites"></a>Voraussetzungen
Die folgenden **Bereiche** sind erforderlich, diese API ausführen: 
## <a name="http-request"></a>HTTP-Anforderung
<!-- { "blockType": "ignored" } -->
```http
POST /workbook/tables(<id|name>)/sort/apply
POST /workbook/worksheets(<id|name>)/tables(<id|name>)/sort/apply

```
## <a name="request-headers"></a>Anforderungsheader
| Name       | Beschreibung|
|:---------------|:----------|
| Autorisierung  | Bearer<code>|


## <a name="request-body"></a>Anforderungstext
Geben Sie im Textkörper Anforderung ein JSON-Objekt mit den folgenden Parametern aus.

| Parameter    | Typ   |Beschreibung|
|:---------------|:--------|:----------|
|Felder|SortField|Die Liste der Bedingungen zu sortieren.|
|matchCase|boolean|Optional Ob die Zeichenfolge Anordnung Schreibweise Auswirkungen haben.|
|Methode|string|Optional Die Sortiermethode für chinesischen Schriftzeichen verwendet.  Mögliche Werte sind: `PinYin`, `StrokeCount`.|

## <a name="response"></a>Antwort
Wenn der Vorgang erfolgreich war, gibt diese Methode `200, OK` Antwortcode. Es werden keine etwas in der Antworttext zurückgegeben.

Es folgt ein Beispiel dafür, wie Sie diese API-aufrufen.
